## Landing Page Message Match

### Quality Score Factors

1. **Expected CTR (35%)**
   - Compelling headlines
   - Relevant ad copy
   - Strong call-to-action
   - Ad extensions used

2. **Ad Relevance (35%)**
   - Keyword in headline
   - Keyword in description
   - Keyword in landing page H1
   - Tight ad group themes

3. **Landing Page Experience (30%)**
   - Fast load time (< 2.5s LCP) ✅
   - Mobile-friendly ✅
   - Relevant content ✅
   - Clear CTA ✅
   - Easy navigation ✅

---

### Message Match Checklist

For each ad, verify:

- [ ] Primary keyword in ad headline
- [ ] Primary keyword in landing page H1
- [ ] Ad promise delivered on landing page
- [ ] CTA matches ad copy
- [ ] Visual elements support message
- [ ] No unexpected content
- [ ] Clear next steps
- [ ] Contact information visible

## Performance Targets

### Key Metrics

| Metric | Target | Acceptable | Poor |
|--------|--------|------------|------|
| **CTR** | > 5% | > 3% | < 2% |
| **Quality Score** | 8-10 | 6-7 | < 6 |
| **Conversion Rate** | > 5% | > 3% | < 2% |
| **CPA** | < £50 | < £75 | > £100 |
| **ROAS** | > 400% | > 300% | < 200% |
