# üá¨üáß UK English Normalisation ‚Äì Regex Rules for Kiro

**Goal:**  
Convert remaining **US English** spellings and phrases in Vivid Media Cheshire content and Google Ads landing pages to **UK English**, while avoiding over-aggressive ‚Äúz ‚Üí s‚Äù conversions.

Apply these rules across:

-   All `page.tsx` files (Services, Hosting, Design, Analytics, Photography, Ads)
    
-   Shared layout components and SEO metadata
    
-   Any JSON/MD content blocks used for copy
    

----------

## üîß Notes for Implementation

-   All regexes below assume **JavaScript/TypeScript-style** syntax.
    
-   Use the **`gi`** flags unless otherwise noted (global, case-insensitive).
    
-   Replacements are **literal UK spellings** ‚Äì no regex in the replacement string.
    
-   Apply rules in the order listed to avoid flip-flopping spellings.
    

----------

## 1. Optimise / Optimisation Family

`// optimize ‚Üí optimise  pattern: /\boptimize\b/gi  replaceWith: 'optimise'  // optimized ‚Üí optimised  pattern: /\boptimized\b/gi  replaceWith: 'optimised'  // optimizing ‚Üí optimising  pattern: /\boptimizing\b/gi  replaceWith: 'optimising'  // optimization ‚Üí optimisation  pattern: /\boptimization\b/gi  replaceWith: 'optimisation'` 

Common phrases to catch on landing pages:

`// speed optimization ‚Üí speed optimisation  pattern: /\bspeed optimization\b/gi  replaceWith: 'speed optimisation'  // performance optimization ‚Üí performance optimisation  pattern: /\bperformance optimization\b/gi  replaceWith: 'performance optimisation'` 

----------

## 2. Analyse / Analysis Family

`// analyze ‚Üí analyse  pattern: /\banalyze\b/gi  replaceWith: 'analyse'  // analyzed ‚Üí analysed  pattern: /\banalyzed\b/gi  replaceWith: 'analysed'  // analyzing ‚Üí analysing  pattern: /\banalyzing\b/gi  replaceWith: 'analysing'` 

----------

## 3. Customise / Personalise

`// customize ‚Üí customise  pattern: /\bcustomize\b/gi  replaceWith: 'customise'  // customized ‚Üí customised  pattern: /\bcustomized\b/gi  replaceWith: 'customised'  // customizing ‚Üí customising  pattern: /\bcustomizing\b/gi  replaceWith: 'customising'  // customization ‚Üí customisation  pattern: /\bcustomization\b/gi  replaceWith: 'customisation'  // personalize ‚Üí personalise  pattern: /\bpersonalize\b/gi  replaceWith: 'personalise'  // personalized ‚Üí personalised  pattern: /\bpersonalized\b/gi  replaceWith: 'personalised'` 

----------

## 4. Behaviour / Colour / Centre / Favourite

`// color ‚Üí colour  pattern: /\bcolor\b/gi  replaceWith: 'colour'  // behavior ‚Üí behaviour  pattern: /\bbehavior\b/gi  replaceWith: 'behaviour'  // center (as noun) ‚Üí centre  pattern: /\bcenter\b/gi  replaceWith: 'centre'  // favorite ‚Üí favourite  pattern: /\bfavorite\b/gi  replaceWith: 'favourite'` 

> ‚ö†Ô∏è If you use ‚Äúcenter‚Äù in CSS classes (e.g. `text-center`), **exclude code blocks** from replacement or run these only over content strings, not class names.

`// organize ‚Üí organise  pattern: /\borganize\b/gi  replaceWith: 'organise'  // organized ‚Üí organised  pattern: /\borganized\b/gi  replaceWith: 'organised'  // organizing ‚Üí organising  pattern: /\borganizing\b/gi  replaceWith: 'organising'  // license (noun) ‚Üí licence  pattern: /\blicense\b/gi  replaceWith: 'licence'  // defense ‚Üí defence  pattern: /\bdefense\b/gi  replaceWith: 'defence'  // inquiry ‚Üí enquiry  pattern: /\binquiry\b/gi  replaceWith: 'enquiry'  // inquiries ‚Üí enquiries  pattern: /\binquiries\b/gi  replaceWith: 'enquiries'`
