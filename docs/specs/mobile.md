# Mobile Optimisation Instructions

## 1. Mobile Priorities
- Mobile is the primary device for Persona A & B.
- Must deliver sub-5 second comprehension.
- CTA visible above the fold at all times.

## 2. Mobile CTA Rules
- Main CTA: **Call Now**
- Sticky CTA: **Call for a Free Ad Plan**
- aria-label: “Call now to get your free, personalised ad plan.”
- Tap targets: minimum 48px height.

## 3. Mobile Layout Requirements
- First 2 lines must state exactly what Joe does.
- No paragraph longer than 3 lines.
- Use bullet points to reduce cognitive load.
- Ensure sticky CTA never overlaps content.

## 4. Mobile Copy Guidelines
- Short, direct, non-jargon.
- Front-load value in every sentence.
- Time-sensitive personas → immediate clarity.

## 5. Mobile Testimonials
Use shortened versions:
- “Joe transformed our social media…”
- “Joe has supported Hampson Auctions for four years…”

## 6. Mobile Speed Requirements
- Lazy-load all below-the-fold images.
- Optimise all images as WebP.
- LCP under 1.8 seconds on 4G.

## 7. Mobile Content Hierarchy
1. Hero heading  
2. One-sentence value statement  
3. Press/proof  
4. CTA  
5. Services  
6. Case study  
7. CTA
